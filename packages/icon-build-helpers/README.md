# ILO Design System - Icon Build Helpers

## Installation

```bash
npm install @ilo-org/icon-build-helpers
```

## Usage

`@ilo-org/icon-build-helpers` is used internally to power the generation of SVG-based projects:

- `@ilo-org/icons`
- `@ilo-org/icons-react`
